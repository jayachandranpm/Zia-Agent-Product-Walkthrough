Agent Editor

A comprehensive 6-tab editor to fully configure every aspect of your AI agent.

Recording

The Configuration tab is your starting point. Set the agent's name, write detailed system prompt instructions, and select the AI provider and model.

  • Agent Name and Description fields
  • System Prompt with personality and behavior guidelines
  • AI Provider selection (Ollama, OpenAI, Anthropic, Gemini)
  • Model selection with dropdowns for each provider

The Workflow tab features a visual drag-and-drop builder for mapping agent decision logic.

  • Add Condition, Action, and Tool nodes to the canvas
  • Connect nodes to define execution flow
  • Clear and reset workflow at any time
  • Visual representation of complex agent logic

The Knowledge tab manages RAG (Retrieval-Augmented Generation) capabilities — upload documents and the agent will reference them during conversations.

  • Upload PDF, DOCX, TXT, MD, and JSON files (up to 25MB each)
  • Track document count, chunks, and total knowledge base size
  • Search indexed knowledge to verify content
  • Automatic chunking and embedding for fast retrieval

The Tools tab gives access to a comprehensive registry of 100+ tools across 25+ Zoho applications.

  • Search and filter by category, Zoho app, or selection status
  • Toggle tools on/off with individual checkboxes
  • Tools include CRM, Desk, Analytics, Books, and more
  • Metrics showing selected vs available tool counts

The Memories tab displays the agent's episodic memory bank (MEMRL), allowing the agent to recall past interactions and context.

  • View stored episodic memories from past interactions
  • Refresh the memory bank to see latest entries
  • Context carryover between conversation sessions

The Settings tab provides advanced configuration toggles for the individual agent.

  • Toggle agent active/inactive status
  • Save agent-specific settings independently

Agent Evaluations

Test and score your agents systematically with automated test cases and real user feedback loops.

Recording

Create automated test suites for your agents — define test inputs, expected outputs, and evaluation metrics. Run them on demand to get pass/fail results with composite scoring.

  • Create evaluation with name, agent, and configurable metrics (Relevance, Accuracy, Tool Usage, Latency)
  • Add multiple test cases with input messages and expected responses
  • Run evaluations on demand — results scored automatically (e.g. 62%, 69%)
  • Dashboard summary: Total Evaluations, Running, Completed, and Avg Score
  • View, re-run, edit, duplicate, or delete any evaluation

Collect real user feedback on agent responses through like/dislike ratings and qualitative reviews. Feedback is tracked per-agent and aggregated into satisfaction metrics.

  • Automatic feedback aggregation from all chat interactions per agent
  • Satisfaction rate calculation (e.g. 33% positive from 3 responses)
  • Performance donut chart showing overall satisfaction visually
  • User Feedback timeline: Positive/Negative tags, reasons, and message previews
  • Export results and generate AI-powered insights for improvement

Automated Evals

Run predefined test cases against agents to measure accuracy, tool usage correctness, and response quality with weighted scoring.

Feedback Evals

Collect human feedback on agent responses with like/dislike ratings, reason tagging, and qualitative reviews.

Scoring Metrics

Track Total Evaluations, Running tests, Completed counts, and Average Score across all evaluation runs.
